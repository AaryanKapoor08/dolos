# Dolos

A financial-crime detection & compliance platform built on the full Java / Spring / Spring-AI
microservices stack.

> Work in progress. Built end-to-end as distributed microservices: real-time transaction
> ingestion and scoring, graph-based fraud-ring detection, event-sourced investigation cases,
> and an AI copilot. Runs entirely locally — zero paid infrastructure.

## Tech (high level)
Java 21 · Spring Boot / Spring Cloud · Spring AI · Kafka · Neo4j · PostgreSQL ·
Axon (CQRS/Event Sourcing) · Drools · BPMN · gRPC · GraphQL · Keycloak · React · Docker / Kubernetes
